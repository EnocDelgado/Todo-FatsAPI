# ToDo API

## Description

This is a Todo RestAPI

## Main Steps
1. **Install Requirements:**

    **Windows:**
    ```
    pip install requirements.txt
    ```
    **MacOS:**
    ```
    pip3 install requirements.txt
    ```

2. **Run Server:**

    ```
    uvicorn app.main:app --reload
    ```
    or
    ```
    uvicorn <folder>.main:app --reload
    ```
